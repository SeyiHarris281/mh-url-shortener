URL Shortener Microservice

URL Shortener

Instructions:

  • Enter [original_url] in URL entry field.

  • Click POST URL.

  • Receive JSON response with format as follows
    • { original_url: "https://www.example.com", short_url: wfu3z }

  • Visit following URL to be redirected to [original_url]
    • [this_project_url]/api/shorturl/[short_url]
